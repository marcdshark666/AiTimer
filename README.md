# AiTimer Free

En futuristisk, gratis webbversion av AiTimer byggd för GitHub Pages.

## Vad som ingår

- neon-inspirerad timer-board med live-klocka
- countdowns och stopwatch med lokal lagring
- fokusläge, historik, presets och snabbkontroller
- ren statisk frontend: `index.html`, `styles.css`, `app.js`

## Publicera med GitHub Pages

1. Pusha `main` till GitHub.
2. Öppna repo-inställningar för GitHub Pages.
3. Välj `Deploy from a branch`.
4. Välj `main` och `/root`.

Efter det publiceras sidan direkt från repots rot.
